## 시간이 없다면 — 15분 코스

**① 목표·원칙 (10분) + ⑥ 플랜의 §2 "데모 6 장면" (5분)** — 이 둘이면 "뭘 왜 만들고, 뭐가 완성인지"는 잡힙니다.

## 한 문장 요약

> 우리는 "위법한 거래는 체결 자체가 안 되는 거래 인프라"를 만들고 있고, 그 증명이 BUIDL 데모 6 장면이며, 지금 필요한 것은 플랜 §4의 결정들입니다.

질문은 단톡에, 구체 수정 제안은 [수정요청서](./appendix-change-request.md) 양식으로 주시면 됩니다.
