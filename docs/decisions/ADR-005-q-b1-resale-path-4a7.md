# ADR-005 — BUIDL 전매(2차 거래) 면제 경로 = §4(a)(7) 주 경로 (Q-B1)

- **상태:** Accepted (2026-06-17) — *single open verification:* general solicitation 판정(보경 변호사)
- **결정자:** 승준(리걸/PM)
- **근거 리서치:** `분석/.../38_BUIDL-전매경로-결정 (4a7 vs 144) — 리서치·권고` (1차 출처 uscode·eCFR·SEC 대조)
- **연계:** Q-B1 · C-00(전매 경로 선택기) · A-03(보경 변호사)

## 1. 배경

사모(Reg D 506(c))로 발행된 BUIDL은 *restricted securities*라, 2차 거래(전매)마다 *별도의 §5 면제*가 필요하다(없으면 매도인이 underwriter로 위법). 후보는 **§4(a)(7)**(상대방 제한: 적격투자자에게만·보유기간 없음)과 **Rule 144**(시간 제한: 비보고 발행자 1년 보유). 내부적으로 Manifest는 Rule 144를, A-03 문서는 §4(a)(7)을 가정해 *불일치*가 있었고, BUIDL 공시는 경로를 명시하지 않는다.

## 2. 결정

**§4(a)(7)을 *주 경로*로, Rule 144를 *보조*(1년 이상 보유한 비-내부자)로 채택한다.**
- `Manifest.resaleFramework = SEC_4A7` (C-00은 보유기간 충족자에 한해 `EITHER`로 144 허용).
- A-03(적격투자자)은 재판매에서 **active**(매수인 AI 게이트).
- A-13(QP)은 §3(c)(7) 때문에 *경로 무관 항상* active(변화 없음).
- §4(a)(1½)(비성문)·Rule 144A(QIB 전용)는 제외.

## 3. 옵션

| 옵션 | 핵심 | 채택 |
|---|---|---|
| **§4(a)(7)** | 보유기간 0 + 적격투자자에게만 + 주법 면제 | ✅ 주 |
| Rule 144 | 1년 보유 후 자유(but DEX 유동성 마비) | 🔵 보조(장기 보유자) |
| §4(a)(1½) | 비성문·사안 판단 | ❌ 자동화 부적합 |
| Rule 144A | QIB 전용 | ❌ 시장 불일치 |

## 4. 근거 (왜 §4(a)(7))

1. **DEX = 유동적 2차 시장** → Rule 144의 *1년 보유기간*은 단기 재매각을 막아 시장을 사실상 마비. §4(a)(7)은 *보유기간 없음.*
2. **§4(a)(7)의 "매수인 AI" 요건은 추가비용 0** — §3(c)(7) 때문에 어차피 매수인 QP(A-13)를 검증하고, AI(A-03)도 이미 구현됨(보경 변호사 작업).
3. **주(州) blue sky 면제**(covered securities) 보너스.

## 5. 결과 (반영)

- **C-00**: `resaleFramework=SEC_4A7` 기본값 · A-03 재판매 active.
- **A-03**(보경 변호사 문서): 재판매 active 버전으로 갱신, "Rule 144 가정" 문구 정정.
- **Manifest 문서**: "Rule 144 가정" → "§4(a)(7) 주 경로".
- **A-12/정보요건**: §4(a)(7) 비보고 발행자 *정보 제공* 조건 구현(발행자/Securitize).

## 6. 열린 질문 / 검증

- 🔴 **general solicitation 판정** — 공개 DEX(whitelist/RFQ 폐쇄 풀)가 §4(a)(7)의 "광고 금지" 요건을 충족하는가. **유일한 blocker, 보경 변호사 확인 대기.** 불충족 시 *Rule 144(1년)+Reg S* 조합으로 후퇴.
- §4(a)(7) 정보 요건의 범위·제공 방식.
- 실제 BUIDL이 의존하는 트랙 확인(Form D·청약서).

## 7. 검증 (close 조건)

보경 변호사가 general solicitation 요건 충족을 확인하면 본 ADR을 *최종 확정(closed)*하고, 위 §5 반영을 일괄 적용한다. 미충족 확인 시 ADR 재작성(Rule 144+Reg S).
